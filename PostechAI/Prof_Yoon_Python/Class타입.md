## Class

* User-defined data type

  : 사용자가 데이터 타입을 정한다.

* 클래스는 사용자가 데이터 타입을 정하는 것

* 클래스 안에 있는 함수(Function)을 메소드(Method)라고 표현한다.

```C
// C - struct 
struct car{
    char color[10];
    int speed;
```

```python
# Python - Class
class Car :
    color = ""
    speed =0
    
    def upSpeed(self,value):
        self.speed+=value
    def downSpeed(self,value):
        self.speed-=value
        
mycar1 = Car()

```
